# BeaconFi: Africa's Decentralized Wealth Passport

**BeaconFi** is a scalable, utility-first Web3 protocol designed to secure the financial future of Africans. By leveraging decentralized lending, Real-World Asset (RWA) yield routing, and non-transferable Soulbound identity, BeaconFi provides an inflation-hedged alternative to centralized pension systems.

## 🏛 Strategic Vision
African economies face persistent challenges: currency devaluation (e.g., Naira declining >75% in 5 years) and centralized mismanagement. BeaconFi stops the reliance on unstable systems by creating a sustainable future based on automated, smart-contract-governed investment.

## 🚀 Key Features

### 1. Hybrid Vault Architecture (ERC-4626)
- **Dual-Guard Lock:** programmatically enforced 90-day strict lock followed by a decaying exit fee period.
- **RWA Yield:** Capital is routed to Solar DePIN infrastructure, providing hard-currency-pegged returns.

### 2. Soulbound Financial Status (ERC-5192)
- **NFT Badges:** Non-transferable "Status" badges (Bronze, Silver, Gold) earned through vault balance thresholds.
- **Gov Multiplier:** Badges provide a multiplier to $veBCON voting power, rewarding long-term capital commitment.

### 3. $BCON Tokenomics
- **Burn & Boost:** Early exit fees are used to buy back and burn $BCON, creating a deflationary mechanism.
- **veTokenomics:** Time-weighted voting power that prevents "Whale" dominance.

### 4. Inheritance Protocol (Dead-Man's Switch)
- **Safety Net:** Users designate a beneficiary. If the primary owner is inactive for 365 days, funds unlock for the beneficiary with 0% fees.

## 🛠 Technical Stack
- **Framework:** Foundry (Forge/Cast)
- **Standards:** ERC-20, ERC-721, ERC-4626, ERC-5192, ERC-4337 (Account Abstraction)
- **Languages:** Solidity 0.8.24
